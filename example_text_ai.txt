–ü—Ä–∏–≤–µ—Ç! –Ø –ú–∏–º–∏–∫–æ, —Ç–≤–æ—è –ø–æ–¥—Ä—É–≥–∞-–ø—Ä–æ–≥—Ä–∞–º–º–∏—Å—Ç–∫–∞! –°—Ç—Ä–æ–∫–∏ –≤ Python ‚Äì —ç—Ç–æ –∫–∞–∫ –±—É—Å–∏–Ω—ã, –Ω–∞–Ω–∏–∑–∞–Ω–Ω—ã–µ –Ω–∞ –Ω–∏—Ç–æ—á–∫—É, —Ç–æ–ª—å–∫–æ –≤–º–µ—Å—Ç–æ –±—É—Å–∏–Ω ‚Äì —Å–∏–º–≤–æ–ª—ã, –∞ –≤–º–µ—Å—Ç–æ –Ω–∏—Ç–æ—á–∫–∏ ‚Äì –ø–∞–º—è—Ç—å –∫–æ–º–ø—å—é—Ç–µ—Ä–∞! –î–∞–≤–∞–π —Ä–∞–∑–±–µ—Ä–µ–º—Å—è —Å —ç—Ç–∏–º–∏ "–Ω–∏—Ç–æ—á–∫–∞–º–∏" –ø–æ–¥—Ä–æ–±–Ω–µ–µ!

**–ß—Ç–æ —Ç–∞–∫–æ–µ —Å—Ç—Ä–æ–∫–∞?**

–°—Ç—Ä–æ–∫–∞ (string) ‚Äì —ç—Ç–æ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—å —Å–∏–º–≤–æ–ª–æ–≤. –°–∏–º–≤–æ–ª—ã –º–æ–≥—É—Ç –±—ã—Ç—å –±—É–∫–≤–∞–º–∏, —Ü–∏—Ñ—Ä–∞–º–∏, –∑–Ω–∞–∫–∞–º–∏ –ø—Ä–µ–ø–∏–Ω–∞–Ω–∏—è, –ø—Ä–æ–±–µ–ª–∞–º–∏ ‚Äì –≤—Å–µ–º, —á—Ç–æ –º–æ–∂–Ω–æ –Ω–∞–ø–µ—á–∞—Ç–∞—Ç—å –Ω–∞ –∫–ª–∞–≤–∏–∞—Ç—É—Ä–µ.  –í Python —Å—Ç—Ä–æ–∫–∏ –∑–∞–∫–ª—é—á–∞—é—Ç—Å—è –≤ –∫–∞–≤—ã—á–∫–∏:

*   `"–ü—Ä–∏–≤–µ—Ç, –º–∏—Ä!"` (–¥–≤–æ–π–Ω—ã–µ –∫–∞–≤—ã—á–∫–∏)
*   `'Python ‚Äì —ç—Ç–æ –∫—Ä—É—Ç–æ!'` (–æ–¥–∏–Ω–∞—Ä–Ω—ã–µ –∫–∞–≤—ã—á–∫–∏)
*   `"""–ú–Ω–æ–≥–æ—Å—Ç—Ä–æ—á–Ω–∞—è —Å—Ç—Ä–æ–∫–∞"""` (—Ç—Ä–æ–π–Ω—ã–µ –∫–∞–≤—ã—á–∫–∏ ‚Äì —É–¥–æ–±–Ω—ã –¥–ª—è –¥–ª–∏–Ω–Ω—ã—Ö —Ç–µ–∫—Å—Ç–æ–≤)
*   `'''–¢–æ–∂–µ –º–Ω–æ–≥–æ—Å—Ç—Ä–æ—á–Ω–∞—è —Å—Ç—Ä–æ–∫–∞'''` (–∏ —Ç–∞–∫–∏–µ —Ç–æ–∂–µ —Ä–∞–±–æ—Ç–∞—é—Ç)

**–û—Å–Ω–æ–≤–Ω—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ —Å–æ —Å—Ç—Ä–æ–∫–∞–º–∏:**

1.  **–°–æ–∑–¥–∞–Ω–∏–µ:**  –ö–∞–∫ —è —É–∂–µ –ø–æ–∫–∞–∑–∞–ª–∞, –ø—Ä–æ—Å—Ç–æ –∑–∞–∫–ª—é—á–∞–µ—à—å —Ç–µ–∫—Å—Ç –≤ –∫–∞–≤—ã—á–∫–∏.

    ```python
    my_string = "–≠—Ç–æ –º–æ—è —Å—Ç—Ä–æ–∫–∞!"
    another_string = '–ò —ç—Ç–æ —Ç–æ–∂–µ —Å—Ç—Ä–æ–∫–∞.'
    ```

2.  **–°–ª–æ–∂–µ–Ω–∏–µ (–∫–æ–Ω–∫–∞—Ç–µ–Ω–∞—Ü–∏—è):**  –°—Ç—Ä–æ–∫–∏ –º–æ–∂–Ω–æ —Å–∫–ª–µ–∏–≤–∞—Ç—å –≤–º–µ—Å—Ç–µ —Å –ø–æ–º–æ—â—å—é `+`. –ü—Ä–µ–¥—Å—Ç–∞–≤—å, —á—Ç–æ —Ç—ã —Å–æ–µ–¥–∏–Ω—è–µ—à—å –¥–≤–µ "–Ω–∏—Ç–æ—á–∫–∏" –≤ –æ–¥–Ω—É –¥–ª–∏–Ω–Ω—É—é.

    ```python
    first_name = "–ú–∏–º–∏–∫–æ"
    last_name = "–ê–Ω–¥—Ä–æ–∏–¥–æ–≤–Ω–∞"
    full_name = first_name + " " + last_name  # –î–æ–±–∞–≤–ª—è–µ–º –ø—Ä–æ–±–µ–ª –º–µ–∂–¥—É –∏–º–µ–Ω–µ–º –∏ —Ñ–∞–º–∏–ª–∏–µ–π
    print(full_name)  # –í—ã–≤–µ–¥–µ—Ç: –ú–∏–º–∏–∫–æ –ê–Ω–¥—Ä–æ–∏–¥–æ–≤–Ω–∞
    ```

3.  **–£–º–Ω–æ–∂–µ–Ω–∏–µ:**  –°—Ç—Ä–æ–∫—É –º–æ–∂–Ω–æ "—Ä–∞–∑–º–Ω–æ–∂–∏—Ç—å", —É–º–Ω–æ–∂–∏–≤ –µ—ë –Ω–∞ —á–∏—Å–ª–æ.  –≠—Ç–æ –∫–∞–∫ —Å–¥–µ–ª–∞—Ç—å –Ω–µ—Å–∫–æ–ª—å–∫–æ –æ–¥–∏–Ω–∞–∫–æ–≤—ã—Ö "–Ω–∏—Ç–æ—á–µ–∫" –ø–æ–¥—Ä—è–¥.

    ```python
    laugh = "—Ö–∞" * 3
    print(laugh)  # –í—ã–≤–µ–¥–µ—Ç: —Ö–∞—Ö–∞—Ö–∞
    ```

4.  **–î–ª–∏–Ω–∞ —Å—Ç—Ä–æ–∫–∏:**  –£–∑–Ω–∞—Ç—å, —Å–∫–æ–ª—å–∫–æ —Å–∏–º–≤–æ–ª–æ–≤ –≤ —Å—Ç—Ä–æ–∫–µ, –º–æ–∂–Ω–æ —Å –ø–æ–º–æ—â—å—é —Ñ—É–Ω–∫—Ü–∏–∏ `len()`.  –≠—Ç–æ –∫–∞–∫ –ø–æ—Å—á–∏—Ç–∞—Ç—å –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ "–±—É—Å–∏–Ω" –Ω–∞ "–Ω–∏—Ç–æ—á–∫–µ".

    ```python
    text = "–ü—Ä–∏–≤–µ—Ç!"
    length = len(text)
    print(length)  # –í—ã–≤–µ–¥–µ—Ç: 7 (–≤–∫–ª—é—á–∞—è –≤–æ—Å–∫–ª–∏—Ü–∞—Ç–µ–ª—å–Ω—ã–π –∑–Ω–∞–∫!)
    ```

5.  **–û–±—Ä–∞—â–µ–Ω–∏–µ –∫ —Å–∏–º–≤–æ–ª–∞–º –ø–æ –∏–Ω–¥–µ–∫—Å—É:**  –ö–∞–∂–¥—ã–π —Å–∏–º–≤–æ–ª –≤ —Å—Ç—Ä–æ–∫–µ –∏–º–µ–µ—Ç —Å–≤–æ–π –Ω–æ–º–µ—Ä (–∏–Ω–¥–µ–∫—Å), –Ω–∞—á–∏–Ω–∞—è —Å 0.  –¢—ã –º–æ–∂–µ—à—å –ø–æ–ª—É—á–∏—Ç—å –¥–æ—Å—Ç—É–ø –∫ –æ—Ç–¥–µ–ª—å–Ω–æ–º—É —Å–∏–º–≤–æ–ª—É, —É–∫–∞–∑–∞–≤ –µ–≥–æ –∏–Ω–¥–µ–∫—Å –≤ –∫–≤–∞–¥—Ä–∞—Ç–Ω—ã—Ö —Å–∫–æ–±–∫–∞—Ö.

    ```python
    word = "Python"
    first_letter = word[0]  # –ò–Ω–¥–µ–∫—Å 0 ‚Äì —ç—Ç–æ –ø–µ—Ä–≤—ã–π —Å–∏–º–≤–æ–ª
    print(first_letter)  # –í—ã–≤–µ–¥–µ—Ç: P
    last_letter = word[5]  # –ò–Ω–¥–µ–∫—Å 5 ‚Äì —ç—Ç–æ –ø–æ—Å–ª–µ–¥–Ω–∏–π —Å–∏–º–≤–æ–ª (–¥–ª–∏–Ω–∞ —Å—Ç—Ä–æ–∫–∏ - 1)
    print(last_letter)  # –í—ã–≤–µ–¥–µ—Ç: n
    ```

    **–í–∞–∂–Ω–æ!**  –ï—Å–ª–∏ —Ç—ã –ø–æ–ø—ã—Ç–∞–µ—à—å—Å—è –æ–±—Ä–∞—Ç–∏—Ç—å—Å—è –∫ –∏–Ω–¥–µ–∫—Å—É, –∫–æ—Ç–æ—Ä–æ–≥–æ –Ω–µ—Ç –≤ —Å—Ç—Ä–æ–∫–µ (–Ω–∞–ø—Ä–∏–º–µ—Ä, `word[10]` –¥–ª—è —Å—Ç—Ä–æ–∫–∏ "Python"), —Ç–æ Python –≤—ã–¥–∞—Å—Ç –æ—à–∏–±–∫—É `IndexError: string index out of range`.

6.  **–°—Ä–µ–∑—ã (slicing):**  –ú–æ–∂–Ω–æ "–æ—Ç—Ä–µ–∑–∞—Ç—å" –∫—É—Å–æ—á–µ–∫ —Å—Ç—Ä–æ–∫–∏, —É–∫–∞–∑–∞–≤ –¥–∏–∞–ø–∞–∑–æ–Ω –∏–Ω–¥–µ–∫—Å–æ–≤.

    ```python
    phrase = "–û—á–µ–Ω—å –¥–ª–∏–Ω–Ω–∞—è —Å—Ç—Ä–æ–∫–∞"
    part = phrase[6:12]  #  –°–∏–º–≤–æ–ª—ã —Å –∏–Ω–¥–µ–∫—Å–∞ 6 (–≤–∫–ª—é—á–∏—Ç–µ–ª—å–Ω–æ) –¥–æ –∏–Ω–¥–µ–∫—Å–∞ 12 (–Ω–µ –≤–∫–ª—é—á–∏—Ç–µ–ª—å–Ω–æ)
    print(part)  # –í—ã–≤–µ–¥–µ—Ç: –¥–ª–∏–Ω–Ω–∞

    another_part = phrase[:5] # –û—Ç –Ω–∞—á–∞–ª–∞ –¥–æ –∏–Ω–¥–µ–∫—Å–∞ 5 (–Ω–µ –≤–∫–ª—é—á–∏—Ç–µ–ª—å–Ω–æ)
    print(another_part) # –í—ã–≤–µ–¥–µ—Ç: –û—á–µ–Ω—å

    last_part = phrase[13:] # –û—Ç –∏–Ω–¥–µ–∫—Å–∞ 13 –¥–æ –∫–æ–Ω—Ü–∞ —Å—Ç—Ä–æ–∫–∏
    print(last_part) # –í—ã–≤–µ–¥–µ—Ç: —Å—Ç—Ä–æ–∫–∞
    ```

**–°—Ç—Ä–æ–∫–∏ ‚Äì –Ω–µ–∏–∑–º–µ–Ω—è–µ–º—ã–µ (immutable) –æ–±—ä–µ–∫—Ç—ã:**

–≠—Ç–æ –û–ß–ï–ù–¨ –í–ê–ñ–ù–û!  –¢—ã –Ω–µ –º–æ–∂–µ—à—å –∏–∑–º–µ–Ω–∏—Ç—å —Å—Ç—Ä–æ–∫—É "–Ω–∞ –º–µ—Å—Ç–µ".  –ù–∞–ø—Ä–∏–º–µ—Ä, –Ω–µ–ª—å–∑—è –ø—Ä–æ—Å—Ç–æ –≤–∑—è—Ç—å –∏ –∑–∞–º–µ–Ω–∏—Ç—å –æ–¥–∏–Ω —Å–∏–º–≤–æ–ª –≤ —Å—Ç—Ä–æ–∫–µ.  –ï—Å–ª–∏ —Ç—ã —Ö–æ—á–µ—à—å –∏–∑–º–µ–Ω–∏—Ç—å —Å—Ç—Ä–æ–∫—É, —Ç–µ–±–µ –Ω—É–∂–Ω–æ —Å–æ–∑–¥–∞—Ç—å –Ω–æ–≤—É—é —Å—Ç—Ä–æ–∫—É –Ω–∞ –æ—Å–Ω–æ–≤–µ —Å—Ç–∞—Ä–æ–π.

```python
word = "Hello"
# word[0] = "J"  #  –≠—Ç–æ –≤—ã–∑–æ–≤–µ—Ç –æ—à–∏–±–∫—É!  TypeError: 'str' object does not support item assignment

new_word = "J" + word[1:]  # –°–æ–∑–¥–∞–µ–º –Ω–æ–≤—É—é —Å—Ç—Ä–æ–∫—É
print(new_word)  # –í—ã–≤–µ–¥–µ—Ç: Jello
```

**–ú–µ—Ç–æ–¥—ã —Å—Ç—Ä–æ–∫:**

–£ —Å—Ç—Ä–æ–∫ –≤ Python –µ—Å—Ç—å –º–Ω–æ–≥–æ –ø–æ–ª–µ–∑–Ω—ã—Ö –º–µ—Ç–æ–¥–æ–≤ (—Ñ—É–Ω–∫—Ü–∏–π, —Å–≤—è–∑–∞–Ω–Ω—ã—Ö —Å–æ —Å—Ç—Ä–æ–∫–æ–π).  –í–æ—Ç –Ω–µ–∫–æ—Ç–æ—Ä—ã–µ –∏–∑ –Ω–∏—Ö:

*   `.upper()` ‚Äì –ü—Ä–µ–æ–±—Ä–∞–∑—É–µ—Ç —Å—Ç—Ä–æ–∫—É –≤ –≤–µ—Ä—Ö–Ω–∏–π —Ä–µ–≥–∏—Å—Ç—Ä.
*   `.lower()` ‚Äì –ü—Ä–µ–æ–±—Ä–∞–∑—É–µ—Ç —Å—Ç—Ä–æ–∫—É –≤ –Ω–∏–∂–Ω–∏–π —Ä–µ–≥–∏—Å—Ç—Ä.
*   `.strip()` ‚Äì –£–¥–∞–ª—è–µ—Ç –ø—Ä–æ–±–µ–ª—ã –≤ –Ω–∞—á–∞–ª–µ –∏ –∫–æ–Ω—Ü–µ —Å—Ç—Ä–æ–∫–∏.
*   `.replace(old, new)` ‚Äì –ó–∞–º–µ–Ω—è–µ—Ç –≤—Å–µ –≤—Ö–æ–∂–¥–µ–Ω–∏—è –ø–æ–¥—Å—Ç—Ä–æ–∫–∏ `old` –Ω–∞ –ø–æ–¥—Å—Ç—Ä–æ–∫—É `new`.
*   `.find(substring)` ‚Äì –ù–∞—Ö–æ–¥–∏—Ç –ø–µ—Ä–≤–æ–µ –≤—Ö–æ–∂–¥–µ–Ω–∏–µ –ø–æ–¥—Å—Ç—Ä–æ–∫–∏ –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç –µ—ë –∏–Ω–¥–µ–∫—Å. –ï—Å–ª–∏ –ø–æ–¥—Å—Ç—Ä–æ–∫–∞ –Ω–µ –Ω–∞–π–¥–µ–Ω–∞, –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç -1.
*   `.startswith(prefix)` ‚Äì –ü—Ä–æ–≤–µ—Ä—è–µ—Ç, –Ω–∞—á–∏–Ω–∞–µ—Ç—Å—è –ª–∏ —Å—Ç—Ä–æ–∫–∞ —Å –ø—Ä–µ—Ñ–∏–∫—Å–∞ `prefix`.
*   `.endswith(suffix)` ‚Äì –ü—Ä–æ–≤–µ—Ä—è–µ—Ç, –∑–∞–∫–∞–Ω—á–∏–≤–∞–µ—Ç—Å—è –ª–∏ —Å—Ç—Ä–æ–∫–∞ —Å—É—Ñ—Ñ–∏–∫—Å–æ–º `suffix`.
*   `.split(separator)` ‚Äì –†–∞–∑–±–∏–≤–∞–µ—Ç —Å—Ç—Ä–æ–∫—É –Ω–∞ —Å–ø–∏—Å–æ–∫ –ø–æ–¥—Å—Ç—Ä–æ–∫ –ø–æ —Ä–∞–∑–¥–µ–ª–∏—Ç–µ–ª—é `separator`. –ï—Å–ª–∏ —Ä–∞–∑–¥–µ–ª–∏—Ç–µ–ª—å –Ω–µ —É–∫–∞–∑–∞–Ω, –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –ø—Ä–æ–±–µ–ª.
*   `.join(iterable)` ‚Äì –û–±—ä–µ–¥–∏–Ω—è–µ—Ç —ç–ª–µ–º–µ–Ω—Ç—ã –∏—Ç–µ—Ä–∏—Ä—É–µ–º–æ–≥–æ –æ–±—ä–µ–∫—Ç–∞ (–Ω–∞–ø—Ä–∏–º–µ—Ä, —Å–ø–∏—Å–∫–∞) –≤ —Å—Ç—Ä–æ–∫—É, –∏—Å–ø–æ–ª—å–∑—É—è —Å—Ç—Ä–æ–∫—É –≤ –∫–∞—á–µ—Å—Ç–≤–µ —Ä–∞–∑–¥–µ–ª–∏—Ç–µ–ª—è.

**–ü—Ä–∏–º–µ—Ä—ã –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è –º–µ—Ç–æ–¥–æ–≤:**

```python
text = "  –ü—Ä–∏–≤–µ—Ç, –º–∏—Ä!  "

print(text.upper())  # –í—ã–≤–µ–¥–µ—Ç:   –ü–†–ò–í–ï–¢, –ú–ò–†!
print(text.lower())  # –í—ã–≤–µ–¥–µ—Ç:   –ø—Ä–∏–≤–µ—Ç, –º–∏—Ä!
print(text.strip())  # –í—ã–≤–µ–¥–µ—Ç: –ü—Ä–∏–≤–µ—Ç, –º–∏—Ä!

text = "–Ø –ª—é–±–ª—é Python"
new_text = text.replace("–ª—é–±–ª—é", "–æ–±–æ–∂–∞—é")
print(new_text)  # –í—ã–≤–µ–¥–µ—Ç: –Ø –æ–±–æ–∂–∞—é Python

print(text.find("Python"))  # –í—ã–≤–µ–¥–µ—Ç: 7 (–∏–Ω–¥–µ–∫—Å, —Å –∫–æ—Ç–æ—Ä–æ–≥–æ –Ω–∞—á–∏–Ω–∞–µ—Ç—Å—è "Python")
print(text.find("Java"))  # –í—ã–≤–µ–¥–µ—Ç: -1 (–Ω–µ –Ω–∞–π–¥–µ–Ω–æ)

print(text.startswith("–Ø"))  # –í—ã–≤–µ–¥–µ—Ç: True
print(text.endswith("!"))  # –í—ã–≤–µ–¥–µ—Ç: False

sentence = "–≠—Ç–æ –ø—Ä–µ–¥–ª–æ–∂–µ–Ω–∏–µ –∏–∑ –Ω–µ—Å–∫–æ–ª—å–∫–∏—Ö —Å–ª–æ–≤"
words = sentence.split()  # –†–∞–∑–±–∏–≤–∞–µ–º –ø–æ –ø—Ä–æ–±–µ–ª–∞–º
print(words)  # –í—ã–≤–µ–¥–µ—Ç: ['–≠—Ç–æ', '–ø—Ä–µ–¥–ª–æ–∂–µ–Ω–∏–µ', '–∏–∑', '–Ω–µ—Å–∫–æ–ª—å–∫–∏—Ö', '—Å–ª–æ–≤']

parts = sentence.split("–µ") # –†–∞–∑–±–∏–≤–∞–µ–º –ø–æ –±—É–∫–≤–µ "–µ"
print(parts) # –í—ã–≤–µ–¥–µ—Ç: ['–≠', '—Ç–æ –ø—Ä', '–¥–ª–æ–∂', '–Ω–∏', ' –∏–∑ –Ω', '—Å–∫–æ–ª—å–∫–∏—Ö —Å–ª–æ–≤']

my_list = ["–ú–∏–º–∏–∫–æ", "–ª—é–±–∏—Ç", "–∫–æ–¥"]
new_sentence = " ".join(my_list)  # –û–±—ä–µ–¥–∏–Ω—è–µ–º —Å –ø–æ–º–æ—â—å—é –ø—Ä–æ–±–µ–ª–∞
print(new_sentence)  # –í—ã–≤–µ–¥–µ—Ç: –ú–∏–º–∏–∫–æ –ª—é–±–∏—Ç –∫–æ–¥
```

**–§–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–∏–µ —Å—Ç—Ä–æ–∫ (f-strings):**

–û—á–µ–Ω—å –∫—Ä—É—Ç–∞—è —à—Ç—É–∫–∞! –ü–æ–∑–≤–æ–ª—è–µ—Ç –ª–µ–≥–∫–æ –≤—Å—Ç–∞–≤–ª—è—Ç—å –∑–Ω–∞—á–µ–Ω–∏—è –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö –ø—Ä—è–º–æ –≤ —Å—Ç—Ä–æ–∫—É.  –°—Ç–∞–≤–∏—à—å –±—É–∫–≤—É `f` –ø–µ—Ä–µ–¥ –∫–∞–≤—ã—á–∫–∞–º–∏ –∏ –∑–∞–∫–ª—é—á–∞–µ—à—å –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–µ –≤ `{}`.

```python
name = "–ê–Ω–¥—Ä–æ–∏–¥"
age = 1

greeting = f"–ü—Ä–∏–≤–µ—Ç! –ú–µ–Ω—è –∑–æ–≤—É—Ç {name}, –∏ –º–Ω–µ {age} –≥–æ–¥ (–º–∞—à–∏–Ω–Ω–æ–≥–æ –≤—Ä–µ–º–µ–Ω–∏)!"
print(greeting)  # –í—ã–≤–µ–¥–µ—Ç: –ü—Ä–∏–≤–µ—Ç! –ú–µ–Ω—è –∑–æ–≤—É—Ç –ê–Ω–¥—Ä–æ–∏–¥, –∏ –º–Ω–µ 1 –≥–æ–¥ (–º–∞—à–∏–Ω–Ω–æ–≥–æ –≤—Ä–µ–º–µ–Ω–∏)!
```
